# Linux User Management Basics

This document covers basic Linux user management tasks commonly performed in IT support and system administration roles.

---

## What is a User in Linux?
A user is an account that allows a person or process to access the Linux system. Each user has a username, user ID (UID), home directory, and permissions.

---

## Create a New User
```bash
sudo adduser username
